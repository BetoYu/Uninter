Protótipos Interativos

App Drogasil

Projeto baseado no aplicativo para celular da farmácia Drogasil.
Possui uma interatividade que demonstra algumas das principais funcionalidades da aplicação original.

App de Tarefas

O projeto foi designado para projetar a interface de um aplicativo, para celular, de lista de tarefas simples.
O objetivo do aplicativo é permitir que os usuários criem uma lista de tarefas, marquem como concluídas e excluam tarefas concluídas.

Jogos

Gatony

Jogo pequeno, criado durante a participação de uma Game Jam.
O projeto foi criado com um tempo de desenvolvimento reduzido devido à dinâmica proposta pela modalidade do evento.

Beyond Nothingness

Pequeno projeto de jogo criado durante o estudo no desenvolvimento de jogos.
Devido à falta de game design progressivamente aplicado, não possui uma progressão amigável para iniciantes.

Sites

Portfólio Pessoal

Site com apresentação de um portifólio pessoal.
O projeto aplica a responsividade e possui detalhes sobre a pessoa abordada, separando cada assunto numa página dedicada.
